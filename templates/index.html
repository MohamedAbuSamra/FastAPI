This is a FastAPI-powered backend for a game store. Most endpoints require authentication via JWT.

Key features:

  • User authentication (/login) with JWT (JSON or form data)
  • Product listing (/products) with pagination and country/location filtering
  • Product details (/products/{product_id})
  • Order creation (/purchase): Purchase a product and receive a full receipt (includes price and product info)
  • Order transactions (/purchase/transaction, /purchase/transactions, /purchase/transactions/{id}): Retrieve or list your orders/receipts
  • User profile (/users/profile): Get your info and transaction count
  • Country listing (/countries): List all available countries
  • CSV import scripts for products and users (see scripts/ folder)
  • Swagger UI for interactive API docs at /docs

Note: All endpoints except /login and /countries require a valid JWT token in the Authorization header.

Quickstart: For development/testing, log in with admin / admin at /login to get a valid JWT token for all endpoints.
